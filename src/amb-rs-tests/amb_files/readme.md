## Legal

Please note that you can't just use the files from the games so you have to "think up" synthetic examples that represent the actual files with the content that doesn't violate other parties' copyright. This doesn't mean you can't write tests for the game files, but make sure that proprietary files don't end up in the repository.

For example, you can look at how AMB files look with a ImHex and [this hexpat](https://github.com/OSA413/Sonic4_Tools/blob/main/imhex-patterns/amb.hexpat), see the pattern (pun intended), and create a file with similar structure and deffects (if any).
